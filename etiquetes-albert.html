10 etiquetas para trabajar con texto

1- < B > o < STRONG > :Sirve para poner la letra en negrita .

2-< I > : Sirve para poner el texto en cursiva

3-< U > : Sirve para subrayar texto .

4-< CODE > : Sirve para escribir un código fuente de un lenguaje de programación .

5-< CITE > : Sirve para colocar texto que a dicho otra persona

6-< SMALL > : Sirve para hacer el texto más reducido.

7-< tt > : Sirve para poner el texto teletipo .

8-< a > :Sirve para añadir un hypervinculo .

9-< dfn > :Sirve para definir un término de una definición

10-< dialog > :Define una caja o una ventana de dialogo .

2 etiquetas para imagenes

1 < img > : Sirve para añadir una imagen .

2 < picture > : Es la version mejorada de < img > ya que permite una mayor flexibidad .

1 etiqueta para enlaces

1- < a href= > :Puedes añadir un enlace i poner información sobre el enlace .

4 etiquetas para ordenar contenido pero que no se vea nada en la pagina

< div > Se emplea para definir un bloque de contenido o sección de página
